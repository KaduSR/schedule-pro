# Schedule Pro 📅

> Sistema de agendamento com calendario drag & drop e sincronia com Google Calendar.

## Stack
React • Google Calendar API • SendGrid • PostgreSQL • OAuth 2.0

## Funcionalidades
- Calendario interativo com drag & drop
- Notificacoes por e-mail (SendGrid)
- Sincronia bidirecional com Google Calendar
- Dashboard de disponibilidade
- Multi-usuario

## Metrica
Sincronia em tempo real com Google Calendar via OAuth 2.0.
